# AHIP DevKit v2

AHIP DevKit provides a reusable installation framework for Allen Hammett
Intelligence Platform packages.

## Features

- Automatic AHIP repository detection
- Manifest-driven package installation
- Backups before replacement
- SHA-256 duplicate detection
- Python syntax validation
- Automatic rollback on failure
- JSON installation logs
- One-click Windows launcher

## Usage

1. Extract the package.
2. Place the extracted folder inside the `infra-signal` repository, or open it
   while your terminal is inside the repository.
3. Double-click `install.bat`.
4. Review the success summary.
